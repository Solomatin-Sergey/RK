Вопрос 1:
Сколькими способами можно перейти в каталог /home/user?
Ответ: есть 2 способа: cd .. и cd -  .
Вопрос 2:
Определить в каких файлах не содержится подстрока.
Ответ: grep -Rlv "подстрока" имя_каталога.
Вопрос 3:
В каталоге /home/user/workspace создать каталог test, в котором будет файл test.txt. Сделать архив, содержащий каталог test.
Ответ: были использованы следующие команды:
mkdir test // создание каталога
cd test // переход в каталог
touch test.txt // создание пустого текстового файла в текущем каталоге
zip -r archive.zip . -i/test // архивирование каталога
Вопрос 4: 
Найти все файлы текущего каталога, в которых встречается подстрока 'aaa' и подстрока 'bbb' одновременно.
Ответ: grep -w "aaa & bbb" имя_текущего_каталога.
Вопрос 5:
Определить количество только каталогов в текущем каталоге.
Ответ: find <каталог_в_котором_ищем> -type d | wc -l .
Вопрос 6:
Определить самые большие по размеру каталоги в системе (относительно /).
Ответ: du -sh /* | sort -h .
